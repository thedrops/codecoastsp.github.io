O que é um hackathon?

O Hackathon é um evento onde desenvolvedores, designers e empreendedores se unem para encontrarem soluções para um desafio proposto, o evento acontece como uma maratona, onde equipes dedicam horas consecutivas para desenvolver a melhor solução dentro do prazo proposto.

Papéis


Idealizadores

Tem uma ideia e deseja transformá-la em um produto viável mesmo sem saber programar, aproveite o evento e junte-se a desenvolvedores para construir um aplicativo.



Desenvolvedores

Está afim de um desafio de desenvolvimento mas está sem ideias? Junte-se a algum idealizador e aceite o desafio de codificar em apenas 24 horas.

Atividades

Curso de Prototipação

Para quem não sabe programar, um curso de prototipação básico será dado para que as ideias possam ser esboçadas de forma mais prática.



Curso de Programação

Para quem quer ir além do protótipo, HTML e CSS serão ensinados para que os projetos tomem forma.



Problemas Matemáticos

Serão propostos desafios do Project Euler e Python challenge para resolução entre as equipes.

Regras dos Desafios

Regras Gerais
  • Os participantes deverão levar suas próprias máquinas e recomenda-se levar adaptadores em T(bejamins) para ligar seus equipamentos;
  • Equipes poderão ter de 2 a 4 membros;
  • O desafio deverá ser completado nas 24 horas propostas;

Regras para Idealizadores
  • Todo projeto deverá ter no mínimo um protótipo funcional;
  • Qualquer ferramenta de prototipação poderá ser utilizada;

Regras para Desenvolvimento
  • Todo projeto deverá ter base em código aberto;
  • Qualquer linguagem de programação poderá ser utilizada;
  • Todo projeto terá de ser mantido em um repositório na ferramenta GitHub e possuir atualizações frequentes para acompanhamento;
